# FacePit - Rede Social Desktop

Este projeto consiste em uma rede social desktop minimalista desenvolvida em Python utilizando uma interface gráfica moderna e persistência de dados local por meio de arquivos JSON.

---

## 🚀 Como Configurar e Executar o Projeto

Como o projeto foi desenvolvido em **Python** (linguagem interpretada), ele não passa por um processo de compilação tradicional (como ocorreria em linguagens como C ou C++). Em vez disso, o código é interpretado e executado diretamente após a instalação das dependências.

Siga os passos abaixo para preparar o ambiente e rodar a aplicação:

### 1. Pré-requisitos
Certifique-se de ter o **Python 3.8 ou superior** instalado no seu sistema operacional.

### 2. Instalação das Dependências
A interface gráfica utiliza a biblioteca `customtkinter` e o processamento de imagens utiliza a biblioteca `Pillow`. Instale as duas dependências rodando o comando abaixo no terminal/prompt de comando:

```bash
pip install customtkinter pillow
